# Jackal Protocol: Actually Decentralized Storage

Jackal is a decentralized, highly-resistant, scalable storage for anyone, anywhere to persist and protect data.

On the Jackal Protocol, you can:

- Build decentralized applications.
- Manage access control and ownership.
- Transfer encrypted data.
- Maintain self custodial data.

... and so much more, without relying on centralized storage providers and other intermediaries.


Jackal isn't like other storage providers. The Jackal Protocol uses both validators and storage providers to provide seamless access to all the [Jackal Protocol Modules]()
